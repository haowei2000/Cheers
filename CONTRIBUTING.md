# Contributing to Cheers

Contributions are welcome — bug reports, documentation, and pull requests
alike. Work branches target `develop`; `main` only accepts merges from
`develop`. Please run `cd server && cargo build && cargo test` and the
frontend build before submitting.

The full contributing guide lives at
[docs/community/CONTRIBUTING.md](docs/community/CONTRIBUTING.md).
