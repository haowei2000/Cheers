## 管理员入口（当前版本）

当前 **没有独立的管理员后台页面**（计划在 M2 提供 Web 管理向导）。管理员与普通用户使用**同一前端地址**，管理操作通过**后端 API** 完成：

| 用途 | 地址 | 说明 |
|------|------|------|
| **聊天前端**（用户与管理员同一入口） | 前端根地址 | Docker：`http://localhost:80`；本地开发：`http://localhost:5173` |
| **API 文档（管理操作用）** | **`http://localhost:8000/docs`** | Swagger 页面，可在此调用创建频道、添加成员等接口；若后端端口不同请替换 8000 |
| **健康检查** | `http://localhost:8000/health` | 检查后端是否正常 |

建项目、加人、加 Bot 等可在前端**左侧「管理」**内完成（创建项目、添加/移除成员），或使用 API 文档。上传文件在频道内输入框旁「上传」；频道上下文在选中频道后底部「频道上下文」。更多入口可通过频道内 **@引导 入口** 查看。
