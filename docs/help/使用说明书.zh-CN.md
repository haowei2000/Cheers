# Cheers 使用说明书

> **语言**：中文 | [English](使用说明书.md)

这是 Cheers 面向开源用户的帮助文档总索引。项目文档分为两类：

- `docs/help/`：当前可用功能的安装、使用、管理和排查说明。
- `docs/arch/`：架构设计、协议与实现约束。若与帮助文档冲突，以帮助文档、README 和当前代码为准。

## 快速开始

1. 按[部署指南](deployment.zh-CN.md)启动服务（Docker Compose、Helm/Kubernetes 或源码运行）。
2. 打开 `http://localhost` 并登录 —— 初始管理员账号在首次启动时由 `.env` 中的 `ADMIN_PASSWORD` 创建。
3. 接入外部智能体 —— 平台没有内置 bot。用 ACP 连接器（`packages/cheers-acp-connector-rs`）运行一个支持 ACP 的智能体（Claude、Codex、OpenCode 等），连接器把它桥接到网关，并通过 `packages/cheers-mcp-server` 提供频道工具。参见[本地Bot配置指南](本地Bot配置指南.zh-CN.md)（主机守护进程方式）或 [AgentBridge接入指南](AgentBridge接入指南.zh-CN.md)（UI 流程与协议）。
4. 把 Bot 加入频道后 `@` 它 —— 例如 `@mybot 怎么用`。

## 开源使用注意事项

- 不要把 `.env.example` 中的示例密码和本地开发密钥用于生产环境。
- 不要提交 `.env`、数据库、日志、上传文件、私钥或 token。
- 文档里的 `cheers.example.com` 是示例域名，请替换成你的实际域名。
- 工作分支 PR 只合并到 `develop`，`main` 只接受来自 `develop` 的合并。
