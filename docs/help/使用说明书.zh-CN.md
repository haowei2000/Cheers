# AgentNexus 使用说明书

> **语言**：中文 | [English](使用说明书.md)

这是 AgentNexus 面向开源用户的帮助文档总索引。项目文档分为两类：

- `docs/help/`：当前可用功能的安装、使用、管理和排查说明。
- `docs/develop/`：架构设计、开发计划和历史方案。若与帮助文档冲突，以帮助文档、README 和当前代码为准。

## 快速开始

1. 按 [安装部署说明](安装部署说明.md) 使用 Docker Compose 启动。
2. 打开 `http://localhost` 进入前端。
3. 使用种子数据时，进入「测试项目」，发送 `@Coordinator 怎么用` 查看引导。
4. 需要接入外部 Agent 时，阅读 [AgentBridge接入指南](AgentBridge接入指南.md)。

## 开源使用注意事项

- 不要把 `.env.example` 中的示例密码和本地开发密钥用于生产环境。
- 不要提交 `.env`、数据库、日志、上传文件、私钥或 token。
- 文档里的 `agentnexus.example.com` 是示例域名，请替换成你的实际域名。
- 工作分支 PR 只合并到 `develop`，`main` 只接受来自 `develop` 的合并。
