> **Language**: English | [中文](使用说明书.zh-CN.md)

This is the public documentation index for Cheers. English is the default language; Chinese mirrors use the `.zh-CN.md` suffix.

## Quick Start

1. Follow the [Installation Guide](安装部署说明.md) to start the stack with Docker Compose.
2. Open `http://localhost`.
3. If seed data is enabled, enter the default channel and send `@Coordinator how do I use this?`.
4. To connect an external Agent, use `/acp-bridge` or read the [Agent Bridge Integration Guide](AgentBridge接入指南.md). ACP-capable local agents are recommended; OpenClaw links are legacy/deprecated.

## Open Source Notes

- Do not use the example secrets in `.env.example` in production.
- Do not commit `.env`, databases, logs, uploads, private keys, or tokens.
- Replace example domains with your own deployment domain.
- Work branch PRs target `develop`; `main` only receives merges from `develop`.
