# AgentNexus 前端操作手册

## 1. 界面概览

AgentNexus 是一个智能协作平台，提供频道聊天、记忆管理、Bot 协作等功能。界面采用现代化的响应式设计，主要分为侧边栏、主聊天区域和底部输入区域。

## 2. 登录与认证

### 2.1 登录界面
- **用户名输入框**：位于登录表单上方，占满整个宽度，输入用户名或邮箱
- **密码输入框**：位于用户名下方，输入登录密码
- **忘记密码链接**：在输入框下方右侧，点击可进入密码重置流程
- **登录按钮**：蓝色背景的"登录"按钮，在登录过程中会显示"处理中..."状态
- **注册选项**：界面底部可切换到注册页面

### 2.2 注册界面
- **用户名输入**：输入想要注册的用户名
- **显示名称输入**：输入个人显示名称
- **密码输入**：输入注册密码
- **邮箱验证**：输入邮箱地址，点击发送验证码
- **注册按钮**：完成信息填写后点击注册

### 2.3 会话管理
- 系统会自动保存登录状态24小时
- 会话过期后需要重新登录

## 3. 主界面布局

### 3.1 侧边栏导航
侧边栏位于界面左侧，包含以下主要功能入口：

#### 顶部工具栏
- **Logo图标**：位于侧边栏顶部，点击可返回主界面
- **搜索图标**：放大镜图标，点击可搜索频道和内容
- **通知图标**：铃铛图标，点击可查看系统通知
- **用户头像/菜单**：右上角显示用户头像，点击可访问个人设置

#### 工作空间面板
- **工作空间列表**：显示用户加入的所有工作空间，每个工作空间名称可点击切换
- **创建工作空间按钮**："+"图标，点击可创建新的工作空间
- **工作空间菜单**：每个工作空间右侧有三个点菜单，可进行管理操作

#### 频道面板
- **频道列表**：显示当前工作空间下的所有频道
- **频道类型标识**：区分公共频道、私有频道等不同类型
- **未读消息指示**：显示频道中的未读消息数量红点
- **创建频道按钮**：频道列表下方的"+"按钮，点击可创建新频道
- **频道菜单**：每个频道右侧有三个点菜单，可进行频道设置

#### 功能面板
- **记忆中心**：五层记忆系统访问入口（ANCHOR、PROGRESS、DECISIONS、FILES_INDEX、RECENT）
- **好友列表**：管理联系人和好友关系
- **管理面板**：系统管理功能（仅管理员可见），包含模型、模板、机器人等管理选项
- **密钥链**：管理敏感信息和API密钥

### 3.2 顶部导航栏
- **频道标题**：显示当前频道名称和类型，点击可进入频道详情
- **频道成员**：显示在线成员数量，点击可查看成员列表
- **频道设置**：齿轮图标，点击可修改频道设置
- **快速连接**：闪电图标，快速连接外部服务
- **频道摘要**：总结图标，生成当前频道摘要

### 3.3 主聊天区域
主聊天区域位于界面中央，显示当前频道的聊天历史：

#### 消息列表
- **消息气泡**：区分用户消息（右侧）和Bot消息（左侧），显示发送者头像、姓名和时间
- **消息状态**：显示消息发送状态（发送中、已发送、已读）
- **消息操作**：鼠标悬停显示回复、引用、复制、删除等操作按钮
- **滚动加载**：向上滚动可加载更多历史消息
- **消息引用**：支持引用其他消息进行回复

#### 附件预览
- **文件卡片**：显示上传文件的预览、名称和类型
- **图片预览**：缩略图展示，点击可放大查看原图
- **文档解析**：显示文档内容摘要和预览

#### 特殊内容块
- **思考块**：用「...」包围的内容可折叠展开
- **代码块**：语法高亮显示代码内容
- **表格**：结构化展示数据
- **表单**：嵌入式表单供用户填写
- **澄清模块**：Bot提出澄清问题的交互组件

### 3.4 底部输入区域
底部输入区域用于发送新消息，包含多个功能组件：

#### 主输入框
- **多行文本框**：支持多行输入，可输入普通消息
- **占位符提示**：显示"发消息到 #[频道名]，@ 呼叫 Bot…"提示
- **自动高度调整**：输入多行内容时会自动扩展高度

#### 输入工具栏
- **密钥链按钮**：钥匙图标，点击显示已保存的密钥列表，可插入到消息中
- **上传文件按钮**：加号图标，点击打开上传菜单
- **上传菜单**：包含上传文件和图片的选项
- **加密模式按钮**：🔒图标，点击切换加密模式，加密状态下消息仅Bot可读原文
- **发送按钮**：
  - 普通模式：绿色"发送"按钮
  - 加密模式：橙色"加密发送"按钮
  - 禁用状态：当输入为空且无附件时为灰色不可点击
- **快捷提示**：显示"Ctrl+Enter"发送提示

#### @提及功能
- **@符号触发**：输入@后自动显示频道内的Bot和用户列表
- **搜索过滤**：可继续输入字符过滤提及列表
- **上下导航**：使用方向键在提及列表中导航
- **选择确认**：按Enter或点击选择提及对象

#### 拖拽上传
- **拖拽覆盖层**：将文件拖拽到聊天区域时显示半透明覆盖层
- **视觉反馈**：显示"可拖拽文件到此处"文字和动画效果
- **支持格式**：提示支持的文件类型（图片：PNG、JPG等；文档：PDF、TXT、MD等）

## 4. 核心功能操作指南

### 4.1 频道管理
#### 创建频道
1. 在当前工作空间下点击"+"按钮
2. 在弹出的表单中输入频道名称
3. 选择频道类型（公共/私有）
4. 点击"保存"完成创建

#### 加入频道
- 在频道列表中点击公共频道名称即可进入

#### 频道设置
- 点击频道名称或右侧齿轮图标进入频道详情页
- 可修改频道名称、描述、类型等信息
- 可管理频道成员，添加或移除用户/Bot

#### 管理频道成员
1. 进入频道设置
2. 点击"成员管理"选项卡
3. 可搜索并添加新成员（用户或Bot）
4. 可移除现有成员

### 4.2 Bot 交互
#### 使用内置Bot
1. 在聊天输入框中输入"@"
2. 从下拉列表中选择要交互的Bot（如@channel bot、@help等）
3. 输入您的问题或指令
4. 点击发送或按Ctrl+Enter发送
5. Bot会在频道中自动回复

#### @ 提及功能
- 输入@后会出现可用的Bot和用户列表
- 使用方向键选择或鼠标点击选择
- 选择后会自动插入提及标签
- Bot检测到提及后会参与对话

#### Bot 详情查看
- 点击Bot消息中的头像或名称可查看Bot详情
- 显示Bot的描述、能力、配置等信息

### 4.3 记忆管理
#### 访问记忆
1. 点击侧边栏的"记忆中心"图标
2. 选择要查看的记忆层级：
   - **ANCHOR（锚点）**：核心目标和约定
   - **PROGRESS（进度）**：任务进展情况
   - **DECISIONS（决策）**：重要决策记录
   - **FILES_INDEX（文件索引）**：文件和资料索引
   - **RECENT（最近关注）**：近期讨论要点

#### 管理记忆
- 在记忆页面可查看、编辑各层级的记忆内容
- 可添加新记忆项，编辑现有项
- Bot可在对话中自动更新相关记忆
- 每个记忆项可添加标题和详细内容

### 4.4 文件上传与管理
#### 上传文件
1. 点击输入框旁的回形针图标（+号图标）
2. 在弹出菜单中选择"上传文件和图片"
3. 选择要上传的文件（支持多种格式）
4. 文件会出现在输入框下方的待发送列表中
5. 点击发送按钮发送文件

#### 拖拽上传
1. 将文件从文件管理器拖拽到聊天区域
2. 出现拖拽覆盖层时释放鼠标
3. 文件会自动上传并在聊天中显示

#### 文件预览
- 点击聊天中的文件卡片可预览内容
- 图片可点击放大查看
- 文档会显示内容摘要
- 支持图片（PNG、JPG、JPEG、WEBP、GIF）、文档（PDF、TXT、MD、DOCX、XLSX）等格式

## 5. 高级功能

### 5.1 工作空间管理
#### 创建工作空间
1. 点击侧边栏顶部的"+"按钮
2. 在弹出的输入框中输入工作空间名称
3. 点击创建完成

#### 工作空间设置
- 点击工作空间名称右侧的三个点菜单
- 可重命名、删除工作空间
- 可管理工作空间成员

#### 邀请成员
1. 进入工作空间设置页面
2. 点击"邀请成员"
3. 输入要邀请的用户邮箱或用户名
4. 发送邀请

### 5.2 个人设置
#### 个人信息编辑
- 点击右上角的用户头像
- 选择"个人资料"选项
- 可编辑显示名称、个人简介等信息
- 点击"保存"应用更改

#### 频道资料设置
- 在当前频道点击右上角的用户头像
- 选择"我在频道的资料"选项
- 可设置频道内的昵称和简介
- 点击"保存"应用更改

### 5.3 通知管理
- 点击侧边栏的通知图标（铃铛图标）
- 查看系统通知和消息提醒
- 可标记为已读或清除通知
- 可设置通知偏好

### 5.4 密钥链管理
- 点击侧边栏的密钥链图标（钥匙图标）
- 查看和管理保存的密钥
- 可添加新的密钥项
- 可删除现有密钥
- 在聊天输入中可通过钥匙按钮插入密钥

### 5.5 管理面板（管理员功能）
只有具有管理员权限的用户才能访问此功能：

#### 模型管理（models标签页）
- **添加模型**：点击"添加模型"按钮配置新的AI模型
- **编辑模型**：点击模型行编辑配置信息
- **启用/禁用**：切换模型的启用状态
- **删除模型**：移除不再需要的模型

#### 模板管理（templates标签页）
- **添加模板**：创建新的提示词模板
- **编辑模板**：修改现有模板内容
- **使用模板**：在创建Bot时选择使用

#### Bot管理（bot标签页）
- **创建Bot**：添加新的Bot实例
- **编辑Bot**：修改Bot配置
- **删除Bot**：移除Bot
- **分配模型/模板**：为Bot指定AI模型和提示词模板

#### LLM设置（llm标签页）
- **配置提供商**：设置LLM服务提供商（如OpenAI、Minimax等）
- **绑定用途**：为不同用途绑定不同的LLM（系统、日志分析、摘要等）
- **测试连接**：验证LLM配置是否有效

#### 用户管理（user标签页）
- **查看用户列表**：查看所有注册用户
- **修改用户角色**：提升或降级用户权限
- **管理用户状态**：激活或禁用用户账户

#### 工作空间管理（workspace标签页）
- **创建工作空间**：为特定用途创建新的工作空间
- **管理成员**：向工作空间添加或移除成员
- **分配资源**：管理工作空间的访问权限

## 6. 快捷键

- **Ctrl+Enter**：在输入框中发送消息
- **Ctrl+K**：快速搜索和切换频道
- **@**：触发Bot/用户提及功能
- **↑/↓箭头**：在@提及列表中导航
- **Esc**：关闭弹窗或取消当前操作
- **Page Up/Page Down**：滚动消息历史

## 7. 故障排除

### 7.1 登录问题
- **问题**：无法登录
- **解决方案**：
  1. 检查用户名和密码是否正确
  2. 确认网络连接正常
  3. 清除浏览器缓存后重试
  4. 检查服务器是否正常运行

### 7.2 消息发送失败
- **问题**：消息无法发送
- **解决方案**：
  1. 检查网络连接
  2. 确认频道权限
  3. 检查输入框是否有内容或附件
  4. 重新刷新页面

### 7.3 Bot 无响应
- **问题**：Bot没有回复
- **解决方案**：
  1. 确认LLM服务配置正确
  2. 检查API连接状态
  3. 确认Bot处于在线状态
  4. 联系系统管理员

### 7.4 文件上传失败
- **问题**：文件无法上传
- **解决方案**：
  1. 检查文件格式是否支持
  2. 确认文件大小不超过限制
  3. 检查网络连接
  4. 重新尝试上传

## 8. 最佳实践

### 8.1 通用操作建议
1. **合理使用记忆功能**：定期整理ANCHOR和DECISIONS层级的内容
2. **有效利用Bot**：在复杂任务中使用@提及合适的Bot协助
3. **文件管理**：上传重要文档到频道，便于团队共享
4. **频道组织**：为不同主题创建专门的频道，保持讨论聚焦

### 8.2 高效沟通技巧
1. **明确提及**：使用@提及确保相关Bot或人员注意到
2. **结构化消息**：使用Markdown格式使消息更易读
3. **引用回复**：使用引用功能保持对话脉络清晰
4. **附件说明**：上传文件时附带简短说明
