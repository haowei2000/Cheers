# AgentNexus 界面交互指南

> **语言**：中文 | [English](<AgentNexus 界面交互指南.md>)

## 1. 界面元素详解

### 1.1 顶部导航栏
- **Logo区域**：显示系统Logo，点击可返回主页
- **频道标题**：显示当前频道名称和类型，点击可进入频道详情
- **频道成员计数**：显示当前频道在线成员数量
- **频道菜单**：右侧三个点图标，点击可访问频道设置、成员管理等功能
- **快速连接按钮**：闪电图标，用于快速连接外部服务
- **摘要生成按钮**：总结图标，生成当前频道对话摘要
- **用户头像**：点击可访问个人设置、频道资料、退出登录等选项

### 1.2 侧边栏组件
#### 工作空间面板
- **工作空间列表**：显示用户加入的所有工作空间
- **工作空间切换**：点击不同工作空间可快速切换
- **创建工作空间**：点击"+"按钮创建新的工作空间
- **工作空间菜单**：右侧三个点图标，提供重命名、删除等选项

#### 频道面板
- **频道列表**：显示当前工作空间下的所有频道
- **频道分类**：公共频道、私有频道等不同类型（通过图标区分）
- **未读消息**：显示频道中的未读消息数量红点
- **创建频道**：点击"+"按钮创建新的频道
- **频道菜单**：每个频道右侧的三个点图标，提供频道设置选项

#### 功能面板
- **记忆中心**：访问五层记忆系统（ANCHOR、PROGRESS、DECISIONS、FILES_INDEX、RECENT）
- **通知中心**：查看系统通知和提醒
- **好友列表**：管理联系人和好友关系
- **密钥链**：管理敏感信息和API密钥
- **管理面板**：系统管理功能（仅管理员可见）

### 1.3 主聊天区域
#### 消息展示区
- **消息气泡**：区分用户消息（右侧，蓝绿色背景）和Bot消息（左侧，蓝色背景）
- **发送者信息**：显示发送者头像、用户名和显示名称
- **发送时间**：精确到分钟的时间戳
- **消息状态**：显示消息发送状态（发送中、已发送、已读）
- **消息操作**：悬停显示回复、引用、复制、删除等操作按钮

#### 附件展示
- **文件卡片**：显示上传文件的预览图标、文件名、类型和大小
- **图片预览**：缩略图展示，点击可放大查看原图
- **文档解析**：显示文档内容摘要和预览
- **下载选项**：可下载查看完整的附件内容

#### 特殊内容块
- **思考块**：用「...」包围的内容可折叠展开，显示为带背景的特殊块
- **代码块**：语法高亮显示代码内容，支持多种编程语言
- **表格**：结构化展示数据，支持滚动查看
- **表单**：嵌入式表单供用户填写，包含各种输入控件
- **澄清模块**：Bot提出澄清问题的交互组件，用户可选择接受/拒绝

### 1.4 底部输入区域
- **多行输入框**：支持多行输入，自动调整高度
- **占位符提示**：显示"发消息到 #[频道名]，@ 呼叫 Bot…"提示
- **@提及功能**：输入@显示可用的Bot和用户列表
- **表情按钮**：可插入表情符号
- **文件上传**：回形针图标，上传文件或图片
- **密钥插入**：钥匙图标，插入预设的密钥
- **加密模式**：🔒图标，切换加密模式
- **发送按钮**：发送消息，颜色随模式变化（普通模式：蓝绿色；加密模式：橙色）

## 2. 交互操作说明

### 2.1 鼠标操作
#### 左键单击
- **导航**：点击侧边栏项目切换界面
- **选择**：选择消息、文件、用户等
- **按钮**：点击界面按钮执行相应功能
- **提及选择**：在@提及列表中选择Bot或用户
- **文件预览**：点击文件卡片预览内容
- **图片放大**：点击图片缩略图查看大图

#### 右键菜单
- **消息**：右键点击消息可显示更多操作选项（复制、引用、删除等）
- **频道**：右键点击频道可显示频道管理选项
- **用户**：右键点击用户可显示用户管理选项
- **文件**：右键点击文件可显示下载等选项

#### 拖拽操作
- **文件上传**：将文件拖拽到聊天区域进行上传
- **界面调整**：拖拽分割线调整界面布局大小
- **消息引用**：拖拽消息到输入框可创建引用

### 2.2 键盘操作
#### 输入操作
- **Enter**：发送消息（需配合Ctrl+Enter）
- **Shift+Enter**：换行而不发送
- **Tab**：在@提及列表中选择项目
- **方向键**：在@提及列表中上下导航
- **Ctrl+A**：全选输入框内容
- **Ctrl+C/V/X**：复制、粘贴、剪切

#### 导航操作
- **Ctrl+K**：快速搜索和切换频道
- **Ctrl+↑/↓**：在消息历史中导航
- **Esc**：关闭弹窗或取消当前操作
- **Ctrl+F**：在聊天记录中搜索内容
- **Page Up/Page Down**：滚动消息历史

### 2.3 触摸操作（移动端）
- **滑动**：上下滑动浏览消息历史
- **长按**：显示消息操作菜单
- **双击**：放大图片预览
- **捏合**：缩放图片预览

## 3. 动画与反馈

### 3.1 加载状态
- **消息发送**：消息气泡显示"发送中"状态
- **Bot响应**：Bot消息显示流式加载动画，文本逐字出现
- **文件上传**：显示上传进度条和百分比
- **API请求**：界面元素显示加载指示器

### 3.2 成功反馈
- **操作成功**：绿色提示消息或成功图标
- **消息发送**：消息状态变为"已发送"或"已读"
- **文件上传**：文件卡片成功显示在聊天中
- **Bot回复**：Bot消息正常显示回复内容

### 3.3 错误反馈
- **操作失败**：红色错误提示消息
- **网络错误**：显示重试选项
- **权限不足**：显示权限错误信息
- **文件上传失败**：显示错误原因和重试选项

## 4. 响应式设计

### 4.1 桌面端
- **宽屏模式**：充分利用屏幕空间，显示完整侧边栏
- **窄屏模式**：自动隐藏部分侧边栏内容，显示更多聊天区域
- **窗口调整**：界面元素自动调整以适应窗口大小

### 4.2 移动端
- **折叠导航**：侧边栏自动折叠为底部导航栏
- **触摸优化**：按钮和交互元素增大便于触摸
- **横屏适配**：横向模式下的界面布局调整
- **手势支持**：支持滑动手势进行页面导航

## 5. 无障碍访问

### 5.1 键盘导航
- 所有功能均可通过键盘操作访问
- 焦点指示器清晰显示当前操作元素
- Tab键顺序逻辑清晰，符合用户预期

### 5.2 屏幕阅读器
- 所有界面元素都有适当的ARIA标签
- 语义化HTML结构便于屏幕阅读器解析
- 动态内容变更及时通知屏幕阅读器

### 5.3 高对比度
- 支持高对比度主题
- 文字与背景有足够的对比度
- 颜色编码有文本替代方案

## 6. 常用交互模式

### 6.1 对话流程
1. **发起对话**：在输入框输入消息内容
2. **选择Bot**：使用@功能选择要交互的Bot
3. **发送消息**：点击发送按钮或使用Ctrl+Enter
4. **等待响应**：观察Bot响应状态，显示加载动画
5. **继续交流**：基于Bot响应继续对话，使用引用功能保持上下文

### 6.2 文件处理流程
1. **上传文件**：点击上传按钮或拖拽文件到聊天区域
2. **处理中**：观察处理进度指示和状态信息
3. **查看结果**：文件处理完成后以卡片形式显示
4. **分享使用**：在对话中引用文件内容或直接分享

### 6.3 记忆管理流程
1. **访问记忆**：点击侧边栏记忆中心图标
2. **选择层级**：选择要查看的记忆层级（ANCHOR、PROGRESS等）
3. **查看内容**：浏览记忆内容，可折叠/展开各个项目
4. **编辑更新**：修改记忆内容（如有权限），点击保存应用更改

运维排查记忆数据时，本地 PostgreSQL 查询必须读取 `.env` 中的 `DATABASE_URL` / `CONTEXT_DB_URL`，不要使用硬编码示例连接信息。

### 6.4 Bot配置流程
1. **提及Bot**：在频道中@特定Bot
2. **发送指令**：发送配置或控制指令
3. **确认响应**：等待Bot确认并执行操作
4. **验证结果**：检查Bot行为是否符合预期

## 7. 界面定制

### 7.1 布局调整
- **侧边栏宽度**：拖拽边缘调整侧边栏宽度
- **聊天区域**：自适应调整主要工作区域
- **分割线**：可拖拽的界面分割线

### 7.2 主题设置
- **明亮主题**：默认白色背景主题
- **暗黑主题**：深色背景，减少眼部疲劳
- **主题切换**：通过设置菜单切换主题

### 7.3 个性化设置
- **显示名称**：在个人资料中设置显示名称
- **个人简介**：设置个人简介信息
- **频道资料**：为不同频道设置不同的昵称和简介

## 8. 性能优化交互

### 8.1 消息加载
- **增量加载**：滚动到顶部时自动加载更多历史消息
- **虚拟滚动**：大量消息时保持流畅滚动
- **延迟渲染**：非可视区域的消息延迟渲染

### 8.2 文件处理
- **压缩上传**：大文件自动压缩上传
- **分片上传**：超大文件分片上传，支持断点续传
- **预览缓存**：文件预览内容缓存，避免重复加载

## 9. 安全相关交互

### 9.1 加密模式
- **启用方式**：点击输入框旁的🔒图标
- **视觉提示**：输入框和发送按钮变为橙色
- **内容保护**：消息内容加密，仅Bot可读原文
- **状态指示**：清楚标识消息为加密状态

### 9.2 密钥管理
- **安全存储**：密钥加密存储在服务器
- **访问控制**：严格控制密钥访问权限
- **使用审计**：记录密钥使用情况

---

*本指南适用于 AgentNexus v1.0 及以上版本的界面交互*
