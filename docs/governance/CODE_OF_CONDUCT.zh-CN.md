# Code of Conduct

> **语言**：中文 | [English](CODE_OF_CONDUCT.md)

Cheers 社区希望保持专业、友善和以问题解决为导向的协作氛围。

## Expected Behavior

- 尊重不同背景、经验水平和观点。
- 讨论聚焦事实、代码、设计和用户影响。
- 给出可复现的问题描述和可验证的技术论据。
- 接受维护者为了项目范围、质量和安全作出的取舍。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视、威胁或持续挑衅。
- 发布他人隐私、凭据、token、漏洞利用细节或未授权数据。
- 在 Issue、PR 或讨论中反复刷屏、跑题或恶意消耗维护资源。

## Enforcement

维护者可以编辑或删除不当内容，关闭讨论，临时或永久限制参与者。安全漏洞披露请遵循 [SECURITY.md](SECURITY.md)。
