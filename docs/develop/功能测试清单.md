## 四、建议测试顺序

1. 启动后端与前端（或 Docker Compose）。
2. 若有需要，执行 `alembic upgrade head` 与 `python -m app.db.seed`（或 SEED_DATA=1）。
3. 按「一」用 Swagger 或 curl 验证 1～13。
4. 按「二」在前端逐项操作验证 14～24。
5. 按「三」确认 25～27。

全部勾选即表示 M1 功能测试通过。
