# 开源发布检查清单

> **语言**：中文 | [English](开源发布检查清单.md)

本文档用于 AgentNexus 首次开源和后续公开 release 前的人工核对。

## 代码与文档

- [ ] 根目录存在 `LICENSE`，README 明确许可证。
- [ ] README 能让新用户在 5 分钟内完成 Docker Compose 启动。
- [ ] `CONTRIBUTING.md`、`SECURITY.md`、`CODE_OF_CONDUCT.md` 已更新。
- [ ] Issue 模板、PR 模板和 Dependabot 配置可用。
- [ ] 文档中的域名、账号、密码和 token 均为示例值。
- [ ] 不包含公司内网地址、私有部署域名、客户名称或内部路径。

## 敏感信息

- [ ] 未提交 `.env`、数据库、日志、上传文件、私钥、证书、生产 token。
- [ ] `.env.example` 中所有密钥均为示例值，并明确提示生产环境必须替换。
- [ ] 已扫描当前工作树中的 `api_key`、`secret`、`token`、`password`、私钥头等关键词。
- [ ] 如仓库曾经包含真实密钥，已轮换密钥并清理 Git 历史。

## 构建与测试

```bash
cd backend && uv run ruff check app/
cd backend && uv run pytest ../tests -v
cd frontend && npm run build
cd packages/agentnexus-bridge-client && npm run lint && npm test && npm run build
cd packages/openclaw-channel-agentnexus && npm run lint && npm test && npm run build
cd packages/agentnexus-acp-connector && npm run lint && npm test && npm run build
```

- [ ] 后端迁移可从空库执行到 head。
- [ ] Docker Compose 可在全新目录按 README 启动。
- [ ] 前端可登录、创建工作区/频道、发送消息、上传文件。
- [ ] Bot / Agent Bridge 最小链路可用，失败时有清晰错误。

## 发布

- [ ] npm 包版本、tag、GitHub Release 名称一致。
- [ ] npm tarball 包含 `dist`、`README.md` 和 `LICENSE`。
- [ ] 容器镜像 tag、release note 和部署文档一致。
- [ ] `main` 只从 `develop` 合并，工作分支 PR 不直接合到 `main`。
